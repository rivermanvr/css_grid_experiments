the fr (fraction) unit

A new unit created for CSS Grid Layout

You can create flexible layouts without using percentages.

it represents a fraction of the available space in the grid container

three flexible responsive columns

with a gap of 20 px

grid-template-columns: 1fr 1fr 1fr;

grid-gap: 20px;

            background-color: rgba(0,95,107,0.8);

            display: grid;
            grid-template-columns: repeat(3, 1fr);
            grid-template-rows: 100px 100px 100px;
            grid-auto-flow: column;
            grid-gap: 20px; }
      

        Puts child 2 and 6 in the grid as defined, but after
        all other items are put in the grid.  Notice where they fall below:

          .gridAuto div:nth-child(2) {
            order: 1;
          }
          
          .gridAuto div:nth-child(6) {
            order: 2;
          }
